# Redberry Store API

**Base URL:** `https://api.redseam.redberryinternship.ge/api`

---

## üîë Authentication

Authentication is optional.  
Endpoints related to user authentication:

### `POST /login`

**Headers**
```http
Accept: application/json
````

**Body**

```json
{
  "email": "hello@redberry.ge",
  "password": "password"
}
```

**Responses**

* `200 OK` ‚Üí Returns user object + token
* `401 Unauthorized` ‚Üí Invalid token
* `422 Unprocessable Entity` ‚Üí Validation error

**Example Response**

```json
{
  "user": {
    "email": "string",
    "name": "string",
    "profile_photo": "https://example.com",
    "id": 1
  },
  "token": "string"
}
```

---

### `POST /register`

**Headers**

```http
Accept: application/json
```

**Body (multipart/form-data)**

* `avatar` (binary) ‚Äì User‚Äôs profile photo
* `email` (string, email)
* `password` (string, password)
* `password_confirmation` (string, password)
* `username` (string)

**Responses**

* `200 OK` ‚Üí User + token
* `401 Unauthorized` ‚Üí Invalid token
* `422 Unprocessable Entity` ‚Üí Validation error

## üõí Products

### `GET /products`

**Query Params**

* `page` (integer) ‚Üí Pagination
* `filter[price_from]` (number) ‚Üí Min price
* `filter[price_to]` (number) ‚Üí Max price
* `sort` (string) ‚Üí Sort by `price` or `created_at` (`-` for descending)

**Responses**

```json
{
  "data": [
    {
      "id": 1,
      "name": "string",
      "release_year": "2025-09-18",
      "image": "https://example.com",
      "price": 1
    }
  ],
  "links": { "first": "...", "last": "...", "prev": "...", "next": "..." },
  "meta": { "current_page": 1, "per_page": 1, "to": 1 }
}
```

---

### `GET /products/{id}`

**Path Params**

* `id` (integer, required)

**Response**

```json
{
  "id": 1,
  "name": "string",
  "description": "string",
  "release_date": "2025-09-18T12:52:46.678Z",
  "cover_image": "https://example.com",
  "images": ["https://example.com"],
  "price": 1,
  "total_price": 1,
  "quantity": 1,
  "brand": { "id": 1, "name": "string", "image": "https://example.com" }
}
```

## üõç Cart

### `POST /cart/products/{product}`

Add product to cart.

**Path Params**

* `product` (integer, required)

**Body**

```json
{
  "color": "red",
  "size": "L",
  "quantity": 2
}
```

**Responses**

* `200 OK` ‚Üí Product added
* `401 Unauthorized`
* `422 Validation Error`

---

### `PATCH /cart/products/{product}`

Update product quantity in cart.

```json
{
  "quantity": 3
}
```

---

### `DELETE /cart/products/{product}`

Removes product from cart.

* `204 No Content`

---

### `GET /cart`

Returns list of cart items.

```json
[
  {
    "id": 1,
    "name": "string",
    "price": 1,
    "quantity": 1,
    "brand": { "id": 1, "name": "string" }
  }
]
```

---

### `POST /cart/checkout`

Checkout all items in cart.

**Responses**

* `200 OK` ‚Üí `{ "message": "Checkout successful" }`
* `400 Bad Request`
* `401 Unauthorized`
